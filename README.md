
# Class Tracker

This is a small class tracker project for keeping track of all classes taken, unmet requirements


## Run Locally

Clone the project

```bash
  git clone https://github.com/dtsivkovski/class-tracker.git
```

Go to the project directory

```bash
  cd class-tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
